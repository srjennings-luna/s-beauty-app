2. The planningItem Schema

Each planning item has the following fields. The ones most relevant to Cowork are flagged.

workingTitle
Short label for the idea. Required. Refine later. Example: "Three Ways · Day 1 · Restlessness".
stream
Radio: Pause & Ponder · Journey · Companion Journey · Tradition Reflection. Drives which sidebar folder the item shows up in.
status
Radio, six steps: Idea → Researching → Drafted → Reviewed → Seeded (live in Sanity) → Archived. Move items left to right as they progress.
journeyArcKey
Dropdown, only visible when stream = Journey. Values: Three Ways, OOTSP, Perelandra, THS. This is what drives the Journeys sub-folders. If Cowork creates a new journey planning item, set this field or the item lands in "Unassigned".
contentType
Matches the nine contentItem.contentType values (sacred-art, photography, thinker, literature, music, food-wine, landscape, watch-listen, math-science). Leave blank for concept-level items.
targetDate / targetMonth
When this content is intended to launch. Leave blank if not yet scheduled.
themes
Reference array to existing theme docs (Light, Silence, Suffering & Beauty, etc.). Optional.
relatedJourney
Reference to a live journey doc. Use when a P&P is meant to warm users up for a specific journey.
relatedPPs
Reference array to live dailyPrompt docs. Cross-stream linking.
feastDayAnchor
Free text for feast day name. Example: "John the Baptist, June 24". Used so the future calendar view can mark feast-anchored slots.
subjectNotes
The lesser-known angle. The editorial idea before any drafting. What is the surprising fact, the dinner-table-test moment, the angle that earns the slot. This is where Cowork's research thesis goes.
sourceMaterial
Citations for verification: book editions, page numbers, Lectio quote provenance. Text-only.
inspirationLinks
NEW. Array of structured external links. Each link has URL (required), Kind (Artwork · Music · Podcast · Video · Article · Book · Pinterest · Other), optional Label, optional Note. Use for podcast episodes that surfaced the idea, candidate artwork pages, Pinterest pins, articles. This is the field Cowork should reach for when capturing research leads.
sheriComments
Sheri's running notes. Reactions to drafts, questions, decisions, redirects. Date entries.
claudeNotes
Claude's research findings, draft hooks, draft contexts, fact-check notes. Cowork reads here; Sheri responds in Sheri Comments.
actionItems
Array. Each: action (required, max 200), owner (radio: Sheri / Claude / Cowork), status (pending / in_progress / done / blocked), optional due date.

4. Editorial Caveats Baked Into claudeNotes

So Cowork does not relitigate these in a future research session, the following caveats are already in the planning items where they apply:

  • Three Ways Day 4 and Day 6 (Pärt). The Cowork journey map paired Arvo Pärt's Spiegel im Spiegel and Tabula Rasa. CLAUDE.md flags Spiegel im Spiegel as a contemplative-app cliché used by every meditation app. Recommend a different Pärt piece (Für Alina, Cantus, or Da pacem Domine) before seeding live. Same caveat applies anywhere Pärt is the default choice.
  • Three Ways Day 7 (Lewis). CLAUDE.md rule: "C.S. Lewis is an internal editorial register test, not an audience-facing name." Do not name Lewis in user-facing copy for this day unless the content item is specifically about him.
  • That Hideous Strength (Open Decision). CLAUDE.md Manual Task #90: does The Abolition of Man stand alone as its own journey, or fold into THS Days 5-6? Resolve before mapping THS days. Marketing positioning: standalone could be aimed at classical educators and former evangelicals.
  • Cross-journey content check (CLAUDE.md, mandatory). Before proposing any artwork, music, or thinker for a planning item, read content-docs/KALLOS-Content-Usage-Log.html first. Flag exact duplicates AND same-artist, same-tradition, same-period as potential duplicates.

5. How Cowork Uses This Surface

When researching a new content item (P&P or Journey day)

  1. Open Sanity Studio. Sidebar, Content Planning, find or create the right item under the right stream.
  2. Set the workingTitle, stream, journeyArcKey (if Journey), status (Idea or Researching), contentType.
  3. Drop the surprising fact / dinner-table angle into subjectNotes. Lead with the fact, not framing.
  4. Add inspiration sources to inspirationLinks: the podcast episode that surfaced the idea, candidate artwork pages, Pinterest pins, articles. Include kind + label + a note on what to come back to.
  5. Put citations and book references in sourceMaterial.
  6. Add an action item if Sheri or Claude needs to do something specific. Set owner.
  7. Move status to Researching, Drafted, etc. as the item progresses.

When proposing a new journey arc

  1. Add a concept-level planningItem with stream = Journey, journeyArcKey set to a new value. Tell Sheri so Claude Code can add the new key to the schema dropdown and add a sidebar folder for it.
  2. Put the arc question, framework, suggested thinkers/composers, suggested artists into subjectNotes and claudeNotes.
  3. Once Sheri approves, draft a day-by-day map (HTML in Documents folder following the format of CONTUERI-Journey-Map-ThreeWays-OOTSP.html).
  4. Claude Code then writes a seed script to convert the day-map into planning items, mirroring the pattern from this seed.

6. Voice and Format Rules That Still Apply

  • No em dashes in any user-facing field (workingTitle, subjectNotes, claudeNotes, action items). Use comma, period, colon, or short hyphen.
  • All scripture quotes use RSV-2CE. If a Lectio quote in the Cowork journey map was drafted in another translation, verify before treating as final.
  • Verified quotes only. Hedged attributions ("commonly attributed," "tradition holds") are apocryphal until proven otherwise. Flag, don't bury.
  • Lewis as internal register test, not audience-facing name. Applies across all journey content.
  • Period context research is mandatory before drafting. The surprising fact almost always lives in what an informed contemporary would understand that a modern reader misses.
  • For the Auditio field, every recommendation must include a sourceable link AND an alternative-register option AND a copy-pasteable Suno or Udio prompt for fallback. Banned defaults: generic "Gregorian chant" without a specific piece, Pärt's Spiegel im Spiegel.
