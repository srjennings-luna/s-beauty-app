Contueri — CC Build Brief

Rebrand: KALLOS → CONTUERI

Created May 2026  ·  Full codebase name rebrand — user-facing text, metadata, share text, localStorage keys
Context: The app is rebranding from KALLOS (Greek for beauty) to CONTUERI (Latin: to contemplate, to gaze upon). This brief covers all code-level changes. Sanity splash content is a separate manual task (noted at end). Do not rename kallos-prompt-streak, kallos-prompt-last, kallos-journey-progress, or kallos-visio-note-* localStorage keys — renaming those would wipe users' streak and progress data.
Pronunciation for splash: kon · too · air · ee

Task 1 — Today Page display name

User-facing · High priority

File: app/page.tsx line 97

Change the visible "KALLOS" text to "CONTUERI".

// After

Also line 30 — the localStorage read for onboarding state:

// Before

// After

The kallos-onboarded → contueri-onboarded rename will cause existing users to see the splash again on next visit. This is intentional — the splash content is changing to reflect the new brand name.
